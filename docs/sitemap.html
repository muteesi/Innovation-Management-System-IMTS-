Strict Role Security Architecture

To comply with security and data policies at Bank of Uganda, cross-role directory access is restricted. Sidebars, headers, and dashboard action buttons are locked to each user's specific folder. No specific user profile should ever view or trigger pages outside their corresponding role bucket.
